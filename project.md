# my-first-project

## Что это за проект

Личный профессиональный веб-проект Светланы Курчевой — AI Builder и основателя WebApp Studio. Проект объединяет персональную цифровую визитку и лендинг веб-студии. Всё создаётся с нуля как первый самостоятельный веб-проект.

## Для кого

- **Автор проекта** — Светлана Курчева, AI Builder
- **Целевая аудитория лендинга** — стартапы, малый и средний бизнес, компании с амбициями роста, которым нужна разработка веб-приложений под ключ

## Страницы, которые уже есть

| Файл | Описание |
|---|---|
| `my-vizitka/vizitka.html` | Персональная цифровая визитка: фото, имя, специализация (typewriter-анимация), ссылка на Telegram |
| `my-landing-1/landing.html` | Лендинг WebApp Studio: hero-блок, преимущества, аудитория, CTA с формой обратной связи |

## Страницы, которые планируются

- **Портфолио / кейсы** — примеры выполненных проектов
- **Страница услуг** — подробное описание всех услуг и цен

## Стиль дизайна

**Тёмная тема** с фиолетовыми акцентами — единый стиль на всех страницах.

- **Фон:** глубокий тёмный градиент (`#08080f`, `#0f0c29`, `#302b63`)
- **Основной акцент:** фиолетовый (`#7c5cfc`, `#7c6fff`, `#a78bfa`)
- **Вторичный акцент:** розовый (`#ff6b8a`) — используется в визитке
- **Карточки:** glassmorphism — полупрозрачный фон с размытием и тонкой рамкой
- **Типографика:** системные шрифты, крупные жирные заголовки с gradient-текстом
- **Анимации:** fade-in, slide-in, typewriter, pulse-glow, IntersectionObserver для появления элементов при скролле
- **Адаптивность:** все страницы адаптированы под мобильные устройства
